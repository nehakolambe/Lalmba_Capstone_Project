# Offline Demo Checklist (Endless OS)

## Setup
1) Start Ollama locally and confirm models are pulled.
2) Run the KB ingest script.
3) Start the Flask backend.
4) Start the React frontend.

## Quick checks
- Register a new user and log in.
- Ask a question about an installed app and verify sources are shown.
- Refresh the page and confirm chat history persists.
- Reset chat and verify history is cleared.

## Sample offline questions (8-10)
1) How do I export a document as PDF in LibreOffice Writer?
2) How do I create a new folder in the Files app?
3) How do I copy a file to another folder?
4) How do I take a screenshot of just one window?
5) How do I open a new tab in Firefox?
6) How do I clear browsing data in Firefox?
7) How do I change the display brightness in Settings?
8) Where do I check storage space on this laptop?
9) How do I insert an image into a LibreOffice document?
10) Where are downloaded files stored?
