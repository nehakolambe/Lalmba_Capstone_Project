# Firefox Browser

Firefox is the web browser for accessing websites and offline web apps.

Common tasks:
- Open a new tab: press **Ctrl+T**.
- Search the web: type in the address bar and press **Enter**.
- Save a page for offline reading: **Menu > Save Page As**.
- Clear browsing data: **Menu > Settings > Privacy & Security > Clear Data**.
- Manage downloads: press **Ctrl+J** to open the Downloads panel.

Tips:
- Use **Private Window** for temporary sessions.
- If a page looks broken, try **Ctrl+F5** to reload.
