# Files (File Manager)

The Files app is used to browse folders, copy files, and organize documents.

Common tasks:
- Open the file manager: click **Files** in the app list.
- Create a folder: right-click and choose **New Folder**.
- Copy files: select a file, press **Ctrl+C**, then **Ctrl+V** in the destination.
- Move files: drag and drop to another folder, or use **Ctrl+X** then **Ctrl+V**.
- Find a file: use the search bar at the top-right.

Tips:
- Use **Home** for personal documents.
- Use **Downloads** for files saved from the browser.
