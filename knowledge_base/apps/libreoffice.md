# LibreOffice Writer

LibreOffice Writer is the word processor used to create and edit documents such as letters, reports, and assignments.

Common tasks:
- Create a new document: Open LibreOffice Writer and choose **New**.
- Save a file: **File > Save** and choose a folder and name.
- Export as PDF: **File > Export As > Export as PDF**.
- Insert a picture: **Insert > Image** and select a file.
- Spell check: **Tools > Spelling** or press **F7**.

Tips:
- Use **Styles** (sidebar) for headings and formatting.
- Press **Ctrl+Z** to undo changes.
